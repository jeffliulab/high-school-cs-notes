    扫描线

    难度：⭐⭐⭐⭐⭐ 前置知识：几何基础、线段树进阶应用、排序

    几何扫描线用一条假想的线扫过平面，在事件点处理几何对象。配合有序集合或线段树，解决矩形面积并、线段交点、最近点对等经典几何问题。

    思想回顾

    把几何对象的「边界」抽象成事件，按某一维（通常 \(x\)）排序，扫描线移动到每个事件时更新「当前活跃对象」的数据结构，从而把二维几何问题降为一维处理。与 扫描线区间查询 同源，这里聚焦几何应用。

    矩形面积并

    求 \(n\) 个轴对齐矩形的并集面积。

    每个矩形 \([x_1,x_2]\times[y_1,y_2]\) 拆成两个事件：

    • 在 \(x_1\) 处，\(y\) 区间 \([y_1,y_2]\) 加 1（进入覆盖）。
    • 在 \(x_2\) 处，\(y\) 区间 \([y_1,y_2]\) 减 1（离开）。

    按 \(x\) 排序事件，扫描线沿 \(x\) 推进。用线段树维护当前 \(y\) 轴上「被覆盖（覆盖次数 ≥ 1）的总长度」。每两个相邻事件 \(x\) 之间，面积贡献 = 覆盖长度 × \(\Delta x\)：

    sort(events by x);
离散化所有 y 坐标;
long long area = 0;
for (int i = 0; i < events.size(); i++) {
    if (i > 0) area += seg.coveredLen() * (events[i].x - events[i-1].x);
    seg.update(events[i].y1, events[i].y2, events[i].type);  // ±1
}

    线段树每个叶子代表一段离散化后的 \(y\) 区间，维护「覆盖次数」和「覆盖长度」。

    最近点对（分治或扫描线）

    平面上 \(n\) 个点，求最近的两点距离。

    扫描线版：按 \(x\) 排序，用 set 按 \(y\) 维护「与当前点 \(x\) 距离 < 当前最优 \(d\)」的候选点，对每个新点只检查 \(y\) 在 \([y-d, y+d]\) 内的少数候选，\(O(n\log n)\)。

    线段相交计数（Bentley–Ottmann 思想）

    扫描线维护当前与扫描线相交的线段（按 \(y\) 排序的有序集合），只检查相邻线段是否相交，高效统计交点。

    事件驱动的统一框架

    扫描线题的通用结构：

    1. 把对象转成事件（进入 / 离开 / 查询）。
    2. 按扫描维排序。
    3. 扫描，用有序集合 / 线段树维护活跃状态。
    4. 在事件处更新答案。

    易错提醒

    • \(y\) 坐标范围大时要离散化后再建线段树。
    • 面积并的线段树维护「覆盖次数 + 覆盖长度」，注意 +1/−1 配对。
    • 相同 \(x\) 的事件处理顺序（先加后算 or 先算后改）要想清楚。
    • 坐标/面积用 long long。

    小结

    • 几何扫描线：对象转事件、按一维排序、扫描时用线段树/有序集合维护活跃状态。
    • 招牌题：矩形面积并（线段树维护覆盖长度）、最近点对、线段相交。
    • 大坐标先离散化；注意事件配对与处理顺序。
