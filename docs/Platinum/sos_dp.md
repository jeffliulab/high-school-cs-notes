# 子集和 DP

> **难度**：⭐⭐⭐⭐⭐
> **前置知识**：[状压 DP](../Gold/bitmask_dp.md)、[位运算入门](../Silver/bitwise.md)

子集和 DP（Sum over Subsets, SOS DP）在 $O(2^n \cdot n)$ 内计算「每个集合的**所有子集**的聚合值」。朴素枚举子集的子集是 $O(3^n)$，SOS 把它优化掉。

## 问题

给定函数 $f(S)$（定义在所有 $2^n$ 个子集上），求：

$$g(S) = \sum_{T \subseteq S} f(T)$$

即每个集合 $S$，统计它所有子集的 $f$ 之和。朴素枚举每个 $S$ 的所有子集是 $\sum_S 2^{|S|} = 3^n$。

## SOS DP 思想

按**维度（位）** 逐步累加。定义中间状态 `dp[S][i]` = 只允许前 $i$ 位与 $S$ 不同的那些子集的 $f$ 之和。逐位转移：第 $i$ 位若在 $S$ 中，可以选择「该位也算上子集变化」。

实际只需一维数组原地更新：

```cpp
// g[S] 初始 = f[S]
for (int i = 0; i < n; i++)
    for (int S = 0; S < (1 << n); S++)
        if (S & (1 << i))
            g[S] += g[S ^ (1 << i)];     // 加上第 i 位为 0 的那一半
```

外层枚举位 $i$，内层枚举所有 $S$；当 $S$ 的第 $i$ 位为 1 时，把「去掉第 $i$ 位」的值累加进来。$O(2^n \cdot n)$。

## 为什么正确

逐位处理保证：处理完第 $i$ 位后，`g[S]` 包含了「与 $S$ 仅在前 $i$ 位的某些 1 上不同」的所有子集贡献。处理完所有 $n$ 位后，`g[S]` 恰好是所有子集之和。这是高维前缀和（每一维做一次前缀和）。

## 超集和（对偶）

求每个集合所有**超集**的和 $\sum_{T \supseteq S} f(T)$，把条件反过来：

```cpp
for (int i = 0; i < n; i++)
    for (int S = 0; S < (1 << n); S++)
        if (!(S & (1 << i)))
            g[S] += g[S | (1 << i)];
```

## 应用

- **子集卷积**、统计「与某掩码兼容/包含的对数」。
- 「对每个二进制掩码，有多少数组元素是它的子集/超集」。
- 配合容斥处理位运算约束的计数问题。

## 例：统计每个掩码的子集中有多少个数组元素

```cpp
for (int x : a) cnt[x]++;            // f[mask] = mask 出现次数
for (int i = 0; i < n; i++)
    for (int S = 0; S < (1<<n); S++)
        if (S & (1<<i)) cnt[S] += cnt[S ^ (1<<i)];
// 现在 cnt[S] = 数组中是 S 的子集的元素个数
```

## 易错提醒

- 外层是**位 $i$**、内层是**掩码 $S$**，顺序不能反。
- 子集和判 `S & (1<<i)`、超集和判 `!(S & (1<<i))`，别搞混。
- $2^n$ 数组，$n$ 一般 $\le 20\sim22$（内存与时间）。
- 累加可能溢出/需取模，按题意处理。

## 小结

- SOS DP 在 $O(2^n n)$ 求每个集合的**所有子集和**（高维前缀和）。
- 模板：外层枚举位、内层枚举掩码，`if (S>>i&1) g[S]+=g[S^(1<<i)]`。
- 超集和把条件取反；用于子集/超集计数、子集卷积等。
