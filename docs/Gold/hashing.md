# 哈希

> **难度**：⭐⭐⭐☆☆
> **前置知识**：[模运算](modular_arithmetic.md)、字符串

哈希把一个对象（尤其是字符串）映射成一个数值，使「比较是否相等」变成 $O(1)$。**字符串哈希**能 $O(1)$ 比较任意两个子串。

## 字符串哈希（多项式哈希）

把字符串看成一个 base 进制数，对大质数取模：

$$h(s) = \big(s_0 \cdot b^{n-1} + s_1 \cdot b^{n-2} + \dots + s_{n-1}\big) \bmod M$$

```cpp
const long long B = 131, MOD = 1e9 + 7;
long long h[N], pw[N];

void build(const string& s) {
    int n = s.size();
    pw[0] = 1;
    for (int i = 1; i <= n; i++) pw[i] = pw[i-1] * B % MOD;
    h[0] = 0;
    for (int i = 0; i < n; i++)
        h[i+1] = (h[i] * B + s[i]) % MOD;     // 前缀哈希
}
```

## O(1) 取子串哈希

子串 $s[l..r]$（0 下标，含端点）的哈希：

```cpp
long long getHash(int l, int r) {            // [l, r]
    return ((h[r+1] - h[l] * pw[r-l+1]) % MOD + MOD) % MOD;
}
```

于是比较两子串是否相等 = 比较两个哈希值，$O(1)$。

## 应用

- **判断两子串相等**：$O(1)$ 比较哈希。
- **找重复子串 / 最长公共子串**：配合二分长度 + 哈希集合。
- **字符串匹配**：滑动比较模式串哈希与文本子串哈希。
- **回文判断**：正向哈希 == 反向哈希。

## 哈希冲突与双哈希

不同字符串可能哈希相同（冲突）。降低概率：

- 用大质数模数和随机 base。
- **双哈希**：用两组 $(B, M)$ 各算一遍，两者都相等才认为相等，冲突概率极低。

```cpp
// 把两个哈希打包成 pair 比较
pair<long long,long long> getHash2(int l, int r){ ... }
```

## unordered_map 的哈希

C++ 的 `unordered_map`/`unordered_set` 基于哈希，平均 $O(1)$。竞赛中要防**被构造数据卡**（针对默认哈希的 TLE），可自定义哈希函数加随机扰动。

## 易错提醒

- 取模后做减法（子串哈希）可能为负，`+MOD` 再取模。
- `pw` 数组要预处理到字符串长度。
- 单哈希有冲突风险，关键题用**双哈希**。
- 前缀哈希数组 `h` 比串长多一位，`h[0]=0`。

## 小结

- 多项式字符串哈希：前缀哈希 + 幂表，$O(1)$ 取任意子串哈希。
- 应用：子串相等比较、重复/最长公共子串、匹配、回文。
- 防冲突用**双哈希**；`unordered_map` 防卡用自定义哈希。
