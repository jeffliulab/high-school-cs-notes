# 位运算入门

> **难度**：⭐⭐☆☆☆
> **前置知识**：二进制表示

位运算直接操作整数的二进制位，速度快、写法简洁。掌握它能高效处理「子集枚举」「状态压缩」等问题，也是 Gold 状压 DP 的基础。

## 基本位运算符

| 运算符 | 名称 | 例（4 位） |
| --- | --- | --- |
| `&` | 与 | `1100 & 1010 = 1000` |
| `\|` | 或 | `1100 \| 1010 = 1110` |
| `^` | 异或 | `1100 ^ 1010 = 0110` |
| `~` | 取反 | `~1100 = ...0011` |
| `<<` | 左移 | `1 << 3 = 1000`（=8） |
| `>>` | 右移 | `1100 >> 2 = 0011` |

## 常用位技巧

```cpp
1 << k          // 2^k，第 k 位为 1 的掩码
x & (1 << k)    // 取出 x 的第 k 位（非 0 即该位为 1）
x | (1 << k)    // 把第 k 位置 1
x & ~(1 << k)   // 把第 k 位清 0
x ^ (1 << k)    // 翻转第 k 位
x & (x - 1)     // 去掉最低位的 1
x & (-x)        // 取出最低位的 1（lowbit，树状数组用）
```

判断第 $k$ 位是否为 1：

```cpp
if ((x >> k) & 1) { /* 第 k 位是 1 */ }
```

## 统计二进制中 1 的个数

```cpp
int cnt = __builtin_popcount(x);    // GCC 内置，O(1)
// 或手动：
int c = 0;
while (x) { x &= (x - 1); c++; }    // 每次去掉一个 1
```

## 用整数表示集合（状态压缩）

一个 `int` 的二进制可表示一个「最多 ~30 个元素的子集」：第 $i$ 位为 1 表示元素 $i$ 在集合中。

```cpp
int S = 0;
S |= (1 << i);          // 加入元素 i
S &= ~(1 << j);         // 删除元素 j
bool has = (S >> i) & 1;// 是否含 i
int sz = __builtin_popcount(S);  // 集合大小
```

## 枚举所有子集

用整数从 $0$ 到 $2^n - 1$ 枚举全部子集：

```cpp
for (int S = 0; S < (1 << n); S++) {
    for (int i = 0; i < n; i++) {
        if ((S >> i) & 1) {
            // 元素 i 在子集 S 中
        }
    }
}
// 共 2^n 个子集，仅 n <= 20 左右可行
```

## 异或的特殊性质

- $x \oplus x = 0$，$x \oplus 0 = x$。
- 满足交换律、结合律。
- 应用：「数组中只有一个数出现奇数次，其余成对」→ 全部异或起来即答案。

```cpp
int ans = 0;
for (int i = 0; i < n; i++) ans ^= a[i];   // 配对的抵消，剩下落单的
```

## 易错提醒

- `1 << k` 当 `k >= 31` 会溢出 `int`，需要 `1LL << k`。
- 运算优先级低：`x & 1 == 0` 实为 `x & (1==0)`，要写 `(x & 1) == 0`。
- `~` 在有符号数上得到负数，注意场景。

## 小结

- 位运算 `& | ^ ~ << >>` 直接操作二进制，快且简洁。
- 常用技巧：取/置/清/翻第 k 位、`x&(x-1)` 去最低位 1、`popcount`。
- 用整数表示集合 → 枚举子集（$2^n$），是状压 DP 的基础；异或可消配对。
