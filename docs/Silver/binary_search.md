# 二分答案

> **难度**：⭐⭐⭐☆☆
> **前置知识**：[有序数组二分](binary_search_sorted.md)

二分答案是银级的标志性技巧：当**直接求最优值困难，但「验证某个值是否可行」容易**时，对答案二分，用判定函数收敛到最优解。

## 适用信号

题目出现「**最大化最小值**」「**最小化最大值**」「求满足条件的最大/最小 X」，且答案具有**单调性**——即：如果 $X$ 可行，那么所有比它更宽松的值也可行（或反之）。

> 例：「使最小间距最大」——间距越小越容易放下，越大越难。存在一个临界值，这正是二分的对象。

## 模板

```cpp
// 求满足 check 的最大 X，check 单调（X 越大越难满足）
bool check(int x) { /* 能否达成，返回 true/false */ }

int lo = MINX, hi = MAXX, ans = -1;
while (lo <= hi) {
    int mid = lo + (hi - lo) / 2;
    if (check(mid)) {
        ans = mid;          // mid 可行，尝试更大
        lo = mid + 1;
    } else {
        hi = mid - 1;       // mid 不可行，往小找
    }
}
// ans 即最大可行值
```

> 若求**最小可行值**，则 `check(mid)` 为真时 `hi = mid - 1` 并记录答案。

## 经典例：最大化最小间距

> 在数轴上 $n$ 个位置中放 $k$ 头牛，使任意两头牛的**最小间距尽可能大**。

判定函数：间距至少为 `d` 时，贪心地从左到右放牛，能否放下 $k$ 头：

```cpp
bool check(int d) {
    int cnt = 1, last = pos[0];      // 第一头放最左
    for (int i = 1; i < n; i++) {
        if (pos[i] - last >= d) {    // 间距够，放一头
            cnt++;
            last = pos[i];
        }
    }
    return cnt >= k;                 // 放下了 k 头 → d 可行
}
// 对 d 二分求最大可行 d（pos 需先排序）
```

## 为什么有效

把「求最优值」转化为「$O(\log(\text{值域}))$ 次判定」。每次判定 $O(n)$，总复杂度 $O(n\log V)$，远优于枚举所有可能答案。

## 浮点二分

答案为实数时，按精度或固定次数迭代：

```cpp
double lo = 0, hi = 1e9;
for (int iter = 0; iter < 100; iter++) {   // 100 次足够精确
    double mid = (lo + hi) / 2;
    if (check(mid)) lo = mid;
    else hi = mid;
}
```

## 易错提醒

- 先确认答案**单调可判定**，否则不能二分。
- `mid` 用 `lo + (hi-lo)/2` 防溢出。
- 想清楚求的是最大还是最小，对应的可行分支不同。
- 二分边界（值域上下界）要覆盖所有可能答案。

## 小结

- 二分答案：把「求最优」转成「判定可行」，要求答案**单调**。
- 信号：最大化最小 / 最小化最大 / 求临界 X。
- 复杂度 $O(n\log V)$；核心是写对 `check` 判定函数。
