# 树算法入门

> **难度**：⭐⭐⭐☆☆
> **前置知识**：[图的遍历](graph_traversal.md)

树是一种特殊的图：$n$ 个点、$n-1$ 条边、连通且**无环**。它结构清晰，遍历时「不走回头路」即可，是 Gold 树形 DP 的基础。

## 树的性质

- $n$ 个顶点恰好 $n-1$ 条边。
- 连通且无环。
- 任意两点间有**唯一路径**。
- 选定一个**根**后，每个非根点有唯一**父节点**，其余相邻点是**子节点**。

## 存树：邻接表

与一般图相同，但因无环，DFS 时只需避免走回父亲：

```cpp
vector<int> adj[100005];
// 无向边 (u, v)
adj[u].push_back(v);
adj[v].push_back(u);
```

## DFS 遍历树（带父节点）

```cpp
void dfs(int u, int parent) {
    for (int v : adj[u]) {
        if (v != parent) {       // 不回到父亲，即不会重复
            dfs(v, u);
        }
    }
}
```

用 `parent` 参数代替 `vis` 数组即可防止往回走（树无环，这就够了）。

## 计算子树大小

后序累加：子树大小 = 1（自己）+ 各子树大小之和：

```cpp
int sz[100005];
void dfs(int u, int p) {
    sz[u] = 1;
    for (int v : adj[u]) {
        if (v != p) {
            dfs(v, u);
            sz[u] += sz[v];      // 加上子节点的子树
        }
    }
}
```

## 计算深度 / 到根距离

前序传递：子节点深度 = 父深度 + 1：

```cpp
int dep[100005];
void dfs(int u, int p) {
    for (int v : adj[u]) {
        if (v != p) {
            dep[v] = dep[u] + 1;
            dfs(v, u);
        }
    }
}
// 调用 dfs(root, 0)，dep[root] = 0
```

## 树的直径（两次 DFS/BFS）

> 树上最远两点的距离。

经典做法：从任意点 BFS 找最远点 $A$，再从 $A$ BFS 找最远点 $B$，$A$ 到 $B$ 的距离即直径。

## 前序 / 后序的区别

- **前序**（进入节点时处理）：适合「从父传给子」的信息，如深度。
- **后序**（处理完所有子节点再处理自己）：适合「从子汇总到父」的信息，如子树大小、子树和。

这正是 Gold [树形 DP](../Gold/dp_on_trees.md) 的核心思路。

## 易错提醒

- 用 `parent` 参数防回头，注意根的父亲设为 0 或 -1。
- 递归深度可达 $n$，链状树（$n=10^5$）可能栈溢出——必要时增栈或改迭代。
- 子树大小/和要在**递归返回后**累加（后序）。
- 树是无向的，加边两个方向。

## 小结

- 树：$n$ 点 $n-1$ 边、连通无环，两点间唯一路径。
- DFS 用 `parent` 参数防回头，无需 `vis`。
- 后序求子树大小/和，前序求深度；两次 BFS 求直径。
