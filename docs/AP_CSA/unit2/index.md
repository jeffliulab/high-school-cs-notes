# 第二单元：选择与迭代

> **考试权重**：25–35%（多选题）

本单元讲**控制流**——如何让程序根据条件做出选择（selection），以及如何重复执行（iteration）。这是算法思维的基础，也是 FRQ Q1 的核心。

## 本单元内容

1. [布尔表达式与关系运算](boolean_expressions.md) —— `boolean` 类型、`< > == != <= >=`。
2. [if 与 if-else 语句](if_statements.md) —— 单分支、双分支、多分支 `if-else-if`。
3. [逻辑运算符与德摩根定律](logical_operators.md) —— `&& || !`、短路求值、德摩根定律。
4. [嵌套条件与多分支](nested_conditionals.md) —— 嵌套 `if`、`switch` 语句。
5. [while 循环](while_loops.md) —— 条件循环、循环不变量、死循环。
6. [for 循环](for_loops.md) —— 计数循环、三段式结构。
7. [嵌套循环](nested_loops.md) —— 双重循环、复杂度直觉。
8. [算法与非正式运行时分析](informal_runtime.md) —— 算法概念、语句执行次数统计。

## 学习目标

- 写出正确的布尔条件，掌握短路求值与德摩根定律。
- 在 `if/else`、`while`、`for` 之间做出恰当选择。
- 理解并避免 off-by-one（差一）错误与死循环。
- 通过统计语句执行次数，非正式地比较算法效率。

## 易错提醒

- `=`（赋值）与 `==`（比较）不要混淆。
- 浮点数不要用 `==` 直接比较。
- `for` 循环边界 `<` 与 `<=` 的差异会造成差一错误。
