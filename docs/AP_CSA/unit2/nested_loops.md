# 嵌套循环

循环里再放循环。外层每执行一次，内层完整跑一轮——这是处理表格、二维数据和「所有配对」的基本工具。

## 基本结构

```java
for (int i = 0; i < 3; i++) {        // 外层
    for (int j = 0; j < 4; j++) {    // 内层
        System.out.print("(" + i + "," + j + ") ");
    }
    System.out.println();
}
```

输出：

```
(0,0) (0,1) (0,2) (0,3)
(1,0) (1,1) (1,2) (1,3)
(2,0) (2,1) (2,2) (2,3)
```

**内层循环体共执行 `3 × 4 = 12` 次**。外层走一步，内层从头到尾跑一遍。

## 执行次数 = 各层次数相乘

这直接关系到效率：两层各 `n` 次 → 共 $n^2$ 次；三层 → $n^3$ 次。数据规模大时，嵌套循环可能**超时**（这是 [算法与非正式运行时分析](informal_runtime.md) 的核心）。

## 打印图形（经典练习）

**直角三角形**——内层次数依赖外层变量：

```java
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print("*");
    }
    System.out.println();
}
```

```
*
**
***
****
*****
```

## 所有配对 / 找重复

嵌套循环可枚举「每一对元素」：

```java
// 统计数组中相等的对数
int count = 0;
for (int i = 0; i < arr.length; i++) {
    for (int j = i + 1; j < arr.length; j++) {   // j 从 i+1 开始避免重复配对
        if (arr[i] == arr[j]) count++;
    }
}
```

> 让内层 `j` 从 `i + 1` 开始，可避免「自己和自己比」以及重复计数（i,j)与(j,i)。

## 二维数组的天然搭档

嵌套循环是遍历二维数组的标准方式（详见 [二维数组遍历](../unit4/array_2d_traversal.md)）：

```java
for (int r = 0; r < grid.length; r++) {
    for (int c = 0; c < grid[0].length; c++) {
        System.out.print(grid[r][c] + " ");
    }
    System.out.println();
}
```

## 易错提醒

- 内外层**循环变量名不能相同**（都用 `i` 会互相覆盖）。
- 注意内层条件是否依赖外层变量（如 `j <= i`）。
- `break` / `continue` 只作用于**最内层**它所在的那个循环。

## 小结

- 外层一次 → 内层完整一轮，总次数为各层相乘。
- 用于打印图形、枚举所有配对、遍历二维数据。
- 嵌套越深复杂度越高，留意超时风险。
