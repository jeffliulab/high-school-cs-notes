# 表达式与赋值

算术表达式如何求值、运算符优先级、以及整数除法与取模这两个高频考点。

## 算术运算符

| 运算符 | 含义 | 例子 | 结果 |
| --- | --- | --- | --- |
| `+` | 加 | `3 + 4` | `7` |
| `-` | 减 | `5 - 8` | `-3` |
| `*` | 乘 | `6 * 7` | `42` |
| `/` | 除 | `7 / 2` | `3`（整数除法） |
| `%` | 取模（余数） | `7 % 2` | `1` |

## 运算优先级

1. 括号 `()`
2. `*`、`/`、`%`（从左到右）
3. `+`、`-`（从左到右）

```java
System.out.println(2 + 3 * 4);    // 14，不是 20
System.out.println((2 + 3) * 4);  // 20
```

不确定时**多加括号**，既正确又易读。

## 整数除法与取模

这是 AP CSA 最爱考的细节。

```java
System.out.println(17 / 5);   // 3  —— 商（向零截断）
System.out.println(17 % 5);   // 2  —— 余数
```

恒等关系：`a == (a / b) * b + (a % b)`。

**经典应用——拆分数位**：

```java
int n = 347;
int ones = n % 10;          // 7（个位）
int tens = (n / 10) % 10;   // 4（十位）
int hundreds = n / 100;     // 3（百位）
```

**判断奇偶 / 整除**：

```java
if (x % 2 == 0) { /* 偶数 */ }
if (year % 4 == 0) { /* 能被 4 整除 */ }
```

> 注意负数取模：在 Java 中 `-7 % 3 == -1`（结果符号跟随被除数）。

## 混合类型运算

表达式中只要有一个 `double`，整个运算就按浮点进行：

```java
System.out.println(5 / 2);      // 2
System.out.println(5.0 / 2);    // 2.5
System.out.println(5 / 2.0);    // 2.5
```

## 赋值

`=` 是**赋值**（把右边的值存入左边变量），不是数学等号。

```java
int x = 10;
x = x + 5;   // 先算右边 10+5=15，再存回 x，x 变为 15
```

`=` 与 `==`（相等比较）完全不同，**切勿混淆**。

## 易错提醒

- `7 / 2` 是 `3` 不是 `3.5`——只要两边都是 `int`。
- 求平均值时 `(a + b) / 2`（int）会先截断；需要小数请用 `(a + b) / 2.0`。
- 大整数相乘可能 `int` 溢出。

## 小结

- 优先级：`() > * / % > + -`。
- 牢记 **整数除法截断** 与 **`%` 取余** 的行为及其拆数位、判奇偶的应用。
- 混合运算中 `double` 会「传染」整个表达式。
